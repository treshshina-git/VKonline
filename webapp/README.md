## Telegram WebApp для запуска VK Live

### Запуск
1. Установить зависимости:
   ```bash
   cd webapp
   npm i
   ```
2. Запустить сервер:
   ```bash
   npm start
   ```

Открытие:
- http://localhost:3000/play?urik=<ch_id>

Внимание: для Telegram WebApp потребуется публичный URL (https) на этот сервер.
